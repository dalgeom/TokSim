# TokSim (톡심) - 기획서

## 1. 서비스 개요

### 한 줄 설명
> 카카오톡 대화를 붙여넣으면 AI가 당신의 말투, 성격, 관계를 분석해주는 웹 서비스

### 타겟 사용자
- 10대 후반 ~ 30대
- 연애/썸/친구 관계에 관심 많은 사람
- 카카오톡을 주로 사용하는 한국인

### 핵심 가치
- **진입장벽 제로**: 설치 없이 웹에서 텍스트 붙여넣기만 하면 끝
- **AI 깊은 분석**: 단순 통계가 아닌 말투/성격/관계 역학 분석
- **공유 욕구 자극**: 결과 카드를 SNS로 공유하고 싶어지는 디자인
- **프라이버시**: 대화 데이터를 서버에 저장하지 않음

## 2. 경쟁사 분석

| 서비스 | 형태 | 장점 | 약점 |
|--------|------|------|------|
| 윤자동 분석기 | 웹 | 무료, 웹 접근 | AI 분석 없음, 단순 통계만 |
| ChatBump AI | 웹 | 고급 AI 분석 | 영어 기반, 유료, 한국 최적화 X |
| 카톡 감정분석 GPT | ChatGPT GPTs | AI 분석 | ChatGPT Plus 필요($20/월) |
| 톡분석 앱 | 앱 | MBTI 분석 | 앱 설치 필요 |
| 연애분석기 앱 | 앱 | 호감도 분석 | 앱 설치 필요 |

### 톡심의 차별점
1. **웹에서 바로** + **무료** + **AI 분석** + **공유 카드** = 이 조합을 가진 서비스가 없음
2. 연애뿐 아니라 친구/직장동료/가족 등 **모든 관계** 분석
3. 한국어 네이티브 UX

## 3. 기능 명세

### Phase 1 — MVP

#### 3.1 랜딩 페이지
- 서비스 소개 (한 줄 카피 + 간단한 비주얼)
- 카카오톡 대화 붙여넣기 텍스트 영역
- "분석하기" 버튼
- 사용법 안내 (카톡에서 대화 내보내기/복사하는 방법)

#### 3.2 카카오톡 대화 파싱
- PC 내보내기 txt 포맷 지원
- 모바일 복사 포맷 지원
- 파싱 결과: { 날짜, 시간, 발신자, 내용 } 배열로 구조화
- 사진/동영상/이모티콘 메시지 처리 (텍스트 없는 메시지)

#### 3.3 기본 통계 분석 (프론트엔드, AI 비용 $0)
- 참여자별 메시지 수 & 비율 (파이 차트)
- 평균 답장 속도 (분 단위)
- 시간대별 대화량 (히트맵 or 바 차트)
- 요일별 대화 패턴
- 가장 많이 쓴 단어 TOP 10
- ㅋ/ㅎ/ㅠ 사용 빈도 & 평균 개수
- 이모티콘/사진 전송 비율
- 가장 긴 대화 vs 가장 짧은 대화
- 대화 시작 비율 (누가 먼저 말을 걸었나)

#### 3.4 AI 분석 (Gemini Flash API)
- 말투 유형 분류: "리액션형", "질문형", "공감형", "팩트형", "유머형" 등
- 대화 온도: 관심도/친밀도를 온도계(0~100)로 시각화
- 관계 역학: "A 주도 / B 반응", "균형잡힌 대화" 등
- 성격 키워드 3개: "#솔직한 #유머러스 #다정한"
- 한 줄 총평: "당신은 읽씹 확률 73%의 위험한 대화 상대입니다"

#### 3.5 결과 페이지
- 기본 통계 시각화 (차트/그래프)
- AI 분석 결과 카드
- 공유용 이미지 카드 생성 (html2canvas)
- 카카오톡 공유 버튼
- 링크 복사 버튼

### Phase 2 — 성장

- Google AdSense 연동
- 모바일 UX 최적화
- 다크모드
- 분석 히스토리 (localStorage)
- SEO 최적화 + 블로그 콘텐츠
- 단체 채팅방 분석 지원
- 인스타그램 스토리 최적화 공유

### Phase 3 — 수익화 확장

- 상세 리포트 (광고 시청 or 소액 결제)
- PDF 리포트 다운로드 (₩1,000~2,000)
- 시계열 비교 ("지난달 vs 이번달 대화 변화")
- 결제 시스템 연동

## 4. 사용자 플로우

```
[1] 랜딩 페이지 방문
     ↓
[2] 카톡 대화 텍스트 붙여넣기 (or 파일 업로드)
     ↓
[3] "분석하기" 클릭
     ↓
[4] 프론트엔드에서 텍스트 파싱 + 기본 통계 계산
     ↓
[5] 서버 엔드포인트로 요약 데이터 전송 → Gemini API 호출
     ↓
[6] 결과 페이지 렌더링
     ├── 기본 통계 차트
     ├── AI 분석 결과
     └── 공유용 카드
     ↓
[7] 카카오톡/인스타/링크 공유 → 바이럴
```

## 5. AI API 호출 전략 (비용 최적화)

### 원칙: 대화 전체를 API에 보내지 않는다

1. 프론트엔드에서 파싱 + 통계 계산
2. API에는 **통계 요약 + 대표 대화 샘플(최대 50개)** 만 전송
3. 프롬프트 예시:
```
다음은 카카오톡 대화 분석 데이터입니다.

[참여자]
- A: 총 342건 메시지, 평균 답장 5분, ㅋ 평균 2.3개, 주요 단어: 맛집, 영화, ㅋㅋㅋ
- B: 총 298건 메시지, 평균 답장 12분, ㅋ 평균 4.1개, 주요 단어: 알겠어, ㅇㅇ, 굿

[대화 샘플]
(최근 대화 20개 + 랜덤 30개)

위 데이터를 분석하여 다음을 JSON으로 응답하세요:
1. 각 참여자의 말투 유형
2. 대화 온도 (0-100)
3. 관계 역학
4. 각 참여자의 성격 키워드 3개
5. 재미있는 한 줄 총평
```

### 예상 토큰 사용량
- 입력: ~2,000 토큰 (요약 + 샘플)
- 출력: ~500 토큰 (JSON 응답)
- Gemini Flash 무료: 하루 250건 충분

## 6. 수익화 전략

### 단기 (MVP)
- **Google AdSense**: 결과 페이지에 배너 광고
- 예상: 일 방문자 1,000명 기준 월 ₩50,000~200,000

### 중기 (Phase 2)
- **프리미엄 리포트**: 광고 시청 후 상세 분석 열람
- **공유 시 무료 잠금 해제**: 공유하면 추가 분석 제공 (바이럴 유도)

### 장기 (Phase 3)
- **PDF 리포트**: ₩1,000~2,000
- **구독 모델**: 월 ₩3,000 무제한 분석

## 7. 바이럴 전략

1. **공유가 곧 마케팅**: 결과 카드 공유 → "나도 해보고 싶다" → 유입
2. **자극적 결과 문구**: "당신은 대화방의 에너지 뱀파이어입니다" 등
3. **비교 유도**: "너도 해봐!" 친구 태그/공유
4. **틱톡/릴스 친화적**: 분석 과정이 화면녹화하기 좋은 UX
5. **커뮤니티 시딩**: 에브리타임, 블라인드, 디시인사이드 등에 자연스럽게 공유
